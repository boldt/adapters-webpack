# Adapters for webpack

## Prepare

```
nvm install
npm install
```

## Build Node Env

```
npm run build-node
```

Test:

```
node dist/node/app.js
```

## Build Browser Env

```
npm run build-browser
npm run build-browser-dev-server
```

Open `dist/index.html` and see the console.

## Clean

```
make clean
```
